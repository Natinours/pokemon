À propos du projet

Collection Pokémon TCG - Épée et Bouclier

Ce site web est dédié à la collection de cartes Pokémon de la série Épée et Bouclier (SWSH1 à SWSH10). Il permet aux collectionneurs de parcourir, rechercher et gérer leur collection de cartes Pokémon en se concentrant sur cette série emblématique.

Fonctionnalités principales

Quelles séries sont disponibles ?

Le site couvre les séries suivantes d'Épée et Bouclier :

  • SWSH1 : Épée et Bouclier
  • SWSH2 : Clash des Rebelles
  • SWSH3 : Ténèbres Embrasées
  • SWSH4 : Voltage Éclatant
  • SWSH5 : Styles de Combat
  • SWSH6 : Règne de Glace
  • SWSH7 : Évolution Céleste
  • SWSH8 : Poing de Fusion
  • SWSH9 : Étoiles Brillantes
  • SWSH10 : Origine Perdue

Comment utiliser le site ?

Le site offre plusieurs fonctionnalités clés :

  • Collection : Parcourez toutes les cartes par série
  • Recherche : Trouvez des cartes spécifiques par nom
  • Filtres : Filtrez les cartes par type (Feu, Eau, etc.)
  • Favoris : Créez votre collection personnelle
  • Images HD : Visualisez les cartes en haute qualité

Caractéristiques techniques

Le site a été développé avec les technologies suivantes :

  • Backend en Go pour des performances optimales
  • Interface utilisateur responsive et moderne
  • Intégration avec l'API TCGdex
  • Système de cache pour un chargement rapide
  • Stockage local pour les favoris

Mises à jour et maintenance

Le site est régulièrement mis à jour pour :

  • Assurer la disponibilité des images des cartes
  • Optimiser les performances de recherche
  • Améliorer l'expérience utilisateur
  • Corriger les bugs éventuels
  • Maintenir la compatibilité avec l'API TCGdex

Remerciements

Ce projet utilise l'API TCGdex pour fournir des données précises et à jour sur les cartes Pokémon. Nous remercions la communauté Pokémon TCG pour son soutien continu.
